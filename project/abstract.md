Project Title: Sentiment Analysis of Tweets 

Team Members: Kapilavai Hamsini 2420030398, Jithyari Sammeta 2420030586 

Section: 5

This project builds a machine learning system to classify tweets as positive, negative, or 
neutral. It uses the Twitter US Airline Sentiment dataset, containing around 14,600 tweets 
directed at major US airlines. The pipeline covers text cleaning (removing URLs, mentions, 
punctuation, and stopwords, then stemming), feature extraction using TF-IDF, and classification 
using Logistic Regression. Model performance is measured using accuracy, precision, recall, 
F1-score, and a confusion matrix. The final model is wrapped into a reusable function that 
predicts sentiment for any new tweet, making the pipeline applicable to real-world use cases 
like brand monitoring and customer feedback analysis.
